# EduFlow AI — School Management Dashboard

A high-fidelity, enterprise-grade SaaS dashboard for school management built with Angular 15.

## Features

- **Left sidebar** (280px) with full navigation and user profile
- **Top navigation** with global search, notifications, AI assistant, dark mode toggle
- **Dashboard** with KPI cards, analytics charts, AI insights, operations widgets, and activity table
- **Floating AI assistant** widget with chat panel
- **Dark mode** support with persisted preference
- **Glassmorphism** design with blue-purple gradient accents

## Quick Start

```bash
cd eduflow-ai
npm install
npm start
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

## Build

```bash
npm run build
```

Output is in `dist/eduflow-ai/`.

## Tech Stack

- Angular 15
- CSS custom properties (design tokens)
- SVG charts (no external chart library)
- Inter font (Google Fonts)
